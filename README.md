## General info
Demo project demonstrate java automation test project using maven and testNg with Shield34.
tests written with POM ( Page Object Model ).


## Technologies
* Java - version 8.0
* Selenium - version 3.8.0
* testng - version 6.8.5

## Setup
Download Project.
```bash
mvn clean install.
```
Run test Class (ExampleTest).
